# AI Candidate Ranking System

An AI-powered candidate ranking platform built using FastAPI, Next.js, Supabase, and semantic embeddings.

## Features

- AI-powered candidate matching
- Semantic similarity ranking
- Modern responsive UI
- Candidate score analysis
- Search and filtering
- Top candidate highlighting
- Supabase database integration
- FastAPI backend APIs
- Next.js frontend

## Tech Stack

### Backend
- FastAPI
- Python
- Sentence Transformers
- Supabase
- pgvector

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
